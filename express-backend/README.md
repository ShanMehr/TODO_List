#Project Backend Setup

##Install dependencies
``` 
# cd express-backend
# used to install all dependencies
npm install
```

## Run the server
```
# cd express-backend
# used to run the server
npm start
```

## Add environment variables
```
# cd express-backend
# touch .env
# Format of .env file
MONGODB_USER=<your mongo username>
MONGDB_PASSWORD=<your mongo password>
```
